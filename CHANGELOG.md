# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-03-17

### Added
- Initial release
- URL to screenshot capture (PNG, JPEG, PDF) with full-page support
- Page metadata extraction: title, description, Open Graph tags, favicon, HTTP status
- Claude Vision optimized: auto-resize to 1568px, token cost estimation
- Viewport presets: desktop (1920x1080), tablet (768x1024), mobile (375x812)
- Element capture via CSS selectors
- Dark mode support (force light/dark color scheme)
- MCP server with 3 tools for Claude Code / MCP clients
- CLI interface
- Python async API
- 29 tests
